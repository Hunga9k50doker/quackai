## ✎ᝰ. RUNNING

- Clone Repository

```bash

cd quackai
```

- Install Dependency

```bash
npm install
```

- Setup config in .env

```bash
nano .env
```

- Setup input value

* proxy: http://user:pass@ip:port

```bash

```
